# Excel Analyse

## Struktur

### Std-Erf
Zentrale Übersicht und Aggregation.

### Mitarbeiter-Blätter
Individuelle Lohnberechnung pro Person.

### Spezialblätter
Unfall, KTG, Tabellen.

## Datenarten

### Stammdaten
- Name

### Bewegungsdaten
- Stunden
- Zuschläge
- Spesen

### Abzüge
- BVG (fixer Betrag)
- Sozialversicherungen

## Einheiten

- Stunden = decimal
- Zuschläge = Stunden
- Spesen = CHF
- BVG = CHF

## Probleme

- Verteilte Logik
- keine zentrale Parametrisierung
- Fehler (#REF)
- Spezialfälle isoliert

## Fazit

Excel ist fachlich wertvoll, aber technisch ungeeignet als Systembasis.
