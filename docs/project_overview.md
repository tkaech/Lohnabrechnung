# Projektuebersicht

Startprojekt fuer eine Schweizer Lohnverwaltung mit:

- C#
- .NET 8
- Avalonia
- SQLite
- Entity Framework Core
- sauber getrennten Schichten

Das aktuelle Geruest zeigt eine minimale Desktop-Anwendung mit Beispielmitarbeitern und vorbereitetem Datenmodell fuer Stundenimport, Spesen und Lohnlaeufe.
