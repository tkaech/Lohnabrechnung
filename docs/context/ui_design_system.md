# UI Design System

## Ziel
Konsistentes Layout und Verhalten.

## Regeln

- einheitliche Abstände
- einheitliche Typografie
- einheitliche Farben
- gleiche Komponenten überall

## Standard Controls

- AppButton
- AppTextBox
- AppNumberBox
- AppDatePicker
- AppComboBox
- AppDataGrid
- AppDialog
- AppFormField
- AppSectionCard

## Prinzip

Keine individuellen Lösungen wenn wiederverwendbare Komponenten möglich sind.
